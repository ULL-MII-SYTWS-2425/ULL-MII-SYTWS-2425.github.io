      # Práctica IAAS

      • Siguiendo las instrucciones en el repositorio SYTW/iaas-ull-es (opens new window) despliegue un ejemplo como el que aparece en crguezl/express-start (opens new window) en su máquina virtual del servicio iaas.ull.es (opens new window).
      • Puede encontrar un vídeo del profesor introduciendo el iaas.ull.es aquí (opens new window).
        • (opens new window)
        • El vídeo es de 2018 y varias de las apps usadas han cambiado de versión pero la esencia de la metodología es la misma
      • Añada en el README.md un pequeño tutorial de como usar y desplegar una aplicación web en iaas.ull.es (opens new window).
        • Haga capturas de pantalla que muestren que su máquina esta bien configurada y funcionando
      • La IP dinámica de su máquina virtual no debería cambiar si no la apaga. Publique la URL de despliegue en su máquina
      • Prepare la máquina para poder trabajar:

        • Instale git si es necesario

        • Configura git (opens new window)

        • Procure que la rama actual aparezca en el prompt de la terminal. Ejemplo para una bash. Pon en el fichero ~/.bash_profile o ~/.bashrc estas líneas:

          parse_git_branch() {
  git branch 2> /dev/null | sed -e '/^[^*]/d' -e 's/* \(.*\)/ (\1)/'
}
export PS1="\u@\h \[\033[32m\]\w\[\033[33m\]\$(parse_git_branch)\[\033[00m\] $ "
          1
          2
          3
          4
          • También puede usar git prompt (opens new window)
          • Puede añadir algo como esto a su PS1: PS1="\$(git branch 2>/dev/null | sed -n 's/* \(.*\)/\1 /p')$ "
          • Recuerde que si su prompt es muy largo siempre puede acortarlo con PROMPT_DIRTRIM=1

        • git aliases (opens new window)

        • Instale GitHub CLI (opens new window) y aprenda a usarla

          • manual (opens new window)

        • Opcionalmente puede instalar linuxbrew (opens new window)

        • Instale nvm (opens new window)

        • Instale nodeJS usando nvm

        • jshint (opens new window) y jshint on vim (opens new window) o JSLint o equivalente (opens new window)

          • Instale NERDTree (opens new window) para vim

        • Instale Express.js

          • Express Web Framework (Node.js/JavaScript) (opens new window) (Mozilla)
